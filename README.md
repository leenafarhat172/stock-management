# 📦 Departmental Stock Management System

A web-based stock management system developed for the
Department of Computer Application and Science.

## 👩‍💻 Developed By
- Name: Leena Farhat
- Department: Computer Application and Science
- Year: 2026

## 📌 About The Project
This system replaces manual register-based stock tracking
with a digital solution. It helps departments manage and
monitor their stock items efficiently.

## ✅ Features
- Secure Login (Admin and Staff roles)
- Dashboard with real-time stock statistics
- Add, Edit and Delete stock items
- Issue stock to persons or departments
- Track all issue records with date and name
- Generate and Print stock reports
- Search items and issue records instantly
- Manage users (Admin only)
- Change password and edit profile

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: Apache (XAMPP)
- Database Tool: phpMyAdmin

## 📁 Project Modules
1. Login Module
2. Dashboard
3. Item Management Module
4. Stock Issue Module
5. Stock Report Module
6. Search Module
7. User Management Module

## ⚙️ How To Run This Project
1. Install XAMPP
2. Clone this repository into htdocs folder
3. Create database named stock_db in phpMyAdmin
4. Import database.sql file
5. Update config.php with your database details
6. Open http://localhost/stock-management/
